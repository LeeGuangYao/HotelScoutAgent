# 003 - 结果列表页规格

## 1. 页面目标

结果列表页用于展示酒店采集结果，帮助用户比较不同平台的列表价、详情页确认价、可信度和截图证据。页面必须强调结果是采集时点信息，不承诺实时有效或全网最低。

## 2. 入口与出口

| 类型 | 说明 |
| --- | --- |
| 路由建议 | `/tasks/:taskId/results` |
| 主要入口 | 任务完成后从任务执行页进入 |
| 次要入口 | 用户主动查看已有任务结果 |
| 返回出口 | 返回 `/tasks/:taskId` 查看任务过程 |
| 外部出口 | 打开酒店来源 URL 或截图证据 |

## 3. 页面布局

```txt
┌──────────────────────────────────────────────┐
│ 顶部：结果标题 + 任务状态 + 返回任务           │
├──────────────────────────────────────────────┤
│ 查询条件摘要 + 结果统计                       │
├──────────────────────────────────────────────┤
│ 筛选排序工具栏                                │
├──────────────────────────────────────────────┤
│ 酒店结果列表                                  │
│ ┌──────────────────────────────────────────┐ │
│ │ 酒店卡片：名称 / 平台 / 价格 / 可信度 / 证据 │ │
│ └──────────────────────────────────────────┘ │
├──────────────────────────────────────────────┤
│ 证据预览弹窗（按需打开）                      │
└──────────────────────────────────────────────┘
```

## 4. 信息结构

### 4.1 顶部结果摘要

展示：

- 标题：比价结果。
- 任务名称。
- 任务状态。
- 采集完成时间。
- 查询条件摘要。
- 结果统计：总结果数、有详情确认价数量、最低详情价、平台数量、证据完整数量。

### 4.2 筛选排序工具栏

字段：

| 控件 | 说明 |
| --- | --- |
| 平台筛选 | 按携程、Booking、飞猪、美团筛选 |
| 可信度筛选 | 高、中、低 |
| 仅看最低价 | 只显示最低详情价结果 |
| 仅看有截图 | 过滤缺少截图证据的结果 |
| 排序方式 | 详情价升序、列表价升序、可信度优先、采集时间 |
| 关键词搜索 | 按酒店名称或位置文本搜索 |

## 5. 酒店结果卡片

每条结果展示：

| 字段 | 说明 |
| --- | --- |
| 酒店名称 | 必填，主标题 |
| 酒店图片 | 可选，MVP 可用占位图 |
| 位置 | 平台页面上采集的位置文本 |
| 平台来源 | 平台标签 |
| 列表页价格 | 从列表页采集的价格，可能为空 |
| 详情页确认价格 | 从详情页确认的价格，用于最低价比较 |
| 价格差异 | 列表价与详情价不一致时展示 |
| 最低价标签 | 仅详情价最低的结果展示 |
| 可信度 | 高、中、低，以及原因 |
| 截图证据 | 查看截图按钮或缺失提示 |
| 来源 URL | 打开来源页面 |
| 采集时间 | 结果生成时间 |

## 6. 价格展示规则

| 场景 | 展示 |
| --- | --- |
| 有列表价和详情价且一致 | 列表价与详情价均正常展示 |
| 有列表价和详情价但不一致 | 详情价高亮，展示“列表价与详情确认价不一致” |
| 只有列表价 | 展示列表价，详情价位置显示“未确认” |
| 只有详情价 | 展示详情价，列表价位置显示“未采集” |
| 价格缺失 | 显示“价格未获取”，不得参与最低价比较 |
| 币种不同 | 显示原币种，MVP 不做汇率换算 |

最低价规则：

1. 只比较存在详情页确认价的结果。
2. 如果多个结果详情价相同，均可标记为“当前最低详情价”。
3. 没有详情页确认价时，不展示最低价标签，只展示无可比较价格说明。

## 7. 可信度展示规则

| 可信度 | 条件示例 | 展示方式 |
| --- | --- | --- |
| 高 | 有详情价、截图、URL、采集时间，页面未异常 | 绿色标签“可信度高” |
| 中 | 详情价或截图有一项缺失，但仍有可追溯 URL | 黄色标签“可信度中” |
| 低 | 仅列表价、缺少截图、页面疑似变化或采集异常 | 红色或灰色标签“可信度低” |

卡片应展示可信度原因，例如：

- “已保存详情页截图”。
- “缺少详情页确认价”。
- “截图证据缺失”。
- “页面价格与列表页不一致”。

## 8. 截图证据交互

### 8.1 有截图

- 展示“查看截图”按钮。
- 点击后打开证据预览弹窗。
- 弹窗展示截图、采集时间、平台、来源 URL、查询条件摘要。

### 8.2 无截图

- 展示“截图缺失”。
- 说明该结果可信度已降低。
- 不提供伪造或占位证据。

## 9. 组件拆分

| 组件 | 职责 |
| --- | --- |
| `ResultsPage` | 页面容器，加载结果和筛选状态 |
| `ResultSummaryStats` | 汇总结果数、最低价、证据完整率 |
| `ResultsToolbar` | 平台、可信度、截图、排序和搜索筛选 |
| `HotelResultList` | 结果列表布局 |
| `HotelResultCard` | 单条酒店结果展示 |
| `PriceFields` | 列表价、详情价、差异说明 |
| `LowestPriceBadge` | 最低详情价标签 |
| `TrustBadge` | 可信度标签与原因 |
| `EvidenceLink` | 截图证据入口和缺失提示 |
| `EvidencePreviewDialog` | 截图预览弹窗 |
| `EmptyResults` | 无结果状态 |

## 10. 空状态与异常状态

| 场景 | 展示策略 |
| --- | --- |
| 结果为空 | 展示“未获取到可展示的酒店结果”，提供返回任务页 |
| 无详情确认价 | 展示“暂无可比较的详情确认价”，不显示最低价 |
| 筛选后为空 | 展示“当前筛选条件下无结果”，提供清除筛选 |
| 证据图片加载失败 | 展示加载失败提示，保留证据元数据 |
| 任务未完成 | 顶部提示“任务仍在运行，结果可能继续更新” |
| 平台全部失败 | 展示平台失败摘要和返回任务页入口 |

## 11. 文案要求

- 最低价标签：当前最低详情价。
- 价格说明：价格为采集时点页面展示信息，可能受税费、会员价、库存和汇率影响。
- 证据说明：截图仅用于证明采集时页面状态，不代表价格长期有效。
- 无结果说明：未获取到可确认价格，不代表平台没有该酒店或没有可订房型。

## 12. 非目标

本页面在第 2 阶段不实现：

- 结果 API 请求。
- 截图文件服务。
- 图片预览代码。
- 价格汇率换算。
- 酒店去重合并算法。
- 前端筛选排序代码。
